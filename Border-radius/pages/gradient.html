90°

O gradiente é uma transição suave entre duas ou mais cores, criando um efeito visual atrativo em design gráfico e web. Ele pode ser aplicado em fundos, botões e elementos de interface.

Como Usar Gradientes no CSS

Você pode usar gradientes como cor de fundo em elementos HTML utilizando a propriedade background no CSS.

Aqui está um exemplo de como usar um gradiente linear:

  
  background: linear-gradient(90deg, #ff7e5f, #feb47b);

E para um gradiente radial:

  
  background: radial-gradient(circle, #ffafbd, #ffc3a0);
