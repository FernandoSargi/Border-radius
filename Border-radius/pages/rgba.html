Sobre Cores HEX e RGBA

Hexadecimal (HEX) é uma forma compacta de representar cores em sistemas digitais. Ele usa seis caracteres (0-9, A-F) para representar valores de cores em RGB (Vermelho, Verde, Azul). Um exemplo é o código #FF5733, que corresponde a um tom de vermelho-alaranjado. Você pode usar o valor HEX em várias propriedades CSS, como background-color, border-color, entre outras.

RGBA (Red, Green, Blue, Alpha) é uma representação de cor com a adição de transparência. O valor de Alpha vai de 0 (totalmente transparente) a 1 (totalmente opaco). Por exemplo, rgba(255, 87, 51, 0.5) é um tom de vermelho-alaranjado com 50% de opacidade.

Exemplo de Uso em Box-Shadow:

Você pode usar tanto o código HEX quanto o RGBA em propriedades como box-shadow. Veja abaixo um exemplo de como aplicar uma sombra com cor HEX e RGBA: 

/* Usando HEX */
.box {
  width: 200px;
  height: 200px;
  background-color: #FF5733;
  box-shadow: 10px 10px 15px #FF5733;
}

/* Usando RGBA */
.box {
  width: 200px;
  height: 200px;
  background-color: rgba(255, 87, 51, 1);
  box-shadow: 10px 10px 15px rgba(255, 87, 51, 0.5);
}

No exemplo acima, a primeira sombra utiliza a cor #FF5733 (HEX), enquanto a segunda utiliza a cor rgba(255, 87, 51, 0.5) (RGBA) com 50% de opacidade.
