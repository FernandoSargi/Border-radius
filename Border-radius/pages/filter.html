Filtros de Imagem CSS

Ajuste os controles para modificar a aparência da imagem. Alterar brilho, contraste e saturação ou aplicar efeitos de sépia e tons de cinza para um visual único. 

  filter: brightness(100%) contrast(100%) saturate(100%) grayscale(0%) sepia(0%);

Exemplos de Propriedades CSS para Filtros de Imagem

brightness(valor) - Ajuste o brilho da imagem.

filter: brightness(150%);

contrast(valor) - Ajuste o contraste da imagem.

filter: contrast(200%);

saturate(valor) - Ajuste a saturação das cores.

filter: saturate(120%);

grayscale(valor) - Converte a imagem para tons de cinza. 

filter: grayscale(50%);

sepia(valor) - Aplica um efeito vintage de sépia. 

filter: sepia(80%);
