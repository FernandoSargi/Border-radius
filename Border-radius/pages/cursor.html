Tipos de Cursor

O CSS oferece uma variedade de cursores que podem ser usados para melhorar a interação do usuário com sua página.

  • default: O cursor padrão.
  • pointer: Usado para links clicáveis.
  • crosshair: Uma cruz (geralmente usado em ferramentas de desenho).
  • text: O cursor de texto, geralmente usado em campos de texto.
  • wait: O cursor que indica que a página está esperando.
