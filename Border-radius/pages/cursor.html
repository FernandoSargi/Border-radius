Efeitos de Cursor

Explore os diferentes tipos de cursores que você pode aplicar a elementos HTML usando CSS.

Selecione um dos efeitos de cursor acima para aplicar ao elemento. O efeito "Pointer" é comum para links, "Crosshair" é usado para seleções precisas, "Texto" é o cursor padrão para texto, e "Esperar" indica que o processo está em andamento.

Exemplo de Código CSS com Cursores

Abaixo estão os exemplos de cursores aplicados aos botões. Cada botão aplica um tipo diferente de cursor ao passar o mouse sobre ele. 

          
            /* Estilos globais */
            * {
              box-sizing: border-box;
            }

            body {
              display: flex;
              flex-direction: column;
              align-items: center;
              margin: 0;
              padding: 20px;
              background-color: #f9f9f9;
              font-family: Arial, sans-serif;
            }

            header {
              text-align: center;
              padding: 20px;
            }

            .button-container {
              display: flex;
              flex-direction: column;
              align-items: center;
            }

            .effect-button {
              margin: 10px;
              padding: 10px 20px;
              background-color: #007bff;
              color: white;
              border: none;
              border-radius: 5px;
              cursor: pointer;
              transition: background-color 0.3s ease;
            }

            .effect-button:hover {
              background-color: #0056b3;
            }

            /* Estilos para os tipos de cursor */
            .effect-button[data-effect="normal"] {
              cursor: default; /* Cursor padrão */
            }

            .effect-button[data-effect="pointer"] {
              cursor: pointer; /* Cursor de link (mãozinha) */
            }

            .effect-button[data-effect="crosshair"] {
              cursor: crosshair; /* Cursor de seleção precisa */
            }

            .effect-button[data-effect="text"] {
              cursor: text; /* Cursor de texto */
            }

            .effect-button[data-effect="wait"] {
              cursor: wait; /* Cursor de espera (reloginho) */
            }

Esses são os tipos de cursores aplicados aos botões. Cada botão tem um efeito de cursor diferente, demonstrando como personalizar o cursor com CSS.
