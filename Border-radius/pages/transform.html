Transformações 3D

0° 0°

Ajuste os controles acima para aplicar rotações no eixo X e Y ao quadrado de transformação. Os valores vão de -180° a 180°, permitindo uma visualização completa da transformação 3D.

transform: rotateX(0deg) rotateY(0deg);

Exemplos de Propriedades CSS - Transform

rotateX(angulo) - Rotaciona o elemento no eixo X. O ângulo pode ser positivo ou negativo. 

transform: rotateX(45deg);

rotateY(angulo) - Rotaciona o elemento no eixo Y. O ângulo pode ser positivo ou negativo. 

transform: rotateY(30deg);

scale(fator) - Altera o tamanho do elemento. Fatores maiores que 1 aumentam o tamanho, e menores que 1 diminuem. 

transform: scale(1.5);

translateX(distância) - Move o elemento horizontalmente. A distância pode ser em pixels ou outras unidades. 

transform: translateX(50px);

translateY(distância) - Move o elemento verticalmente. 

transform: translateY(-20px);

skewX(angulo) - Inclina o elemento no eixo X. O ângulo pode ser positivo ou negativo. 

transform: skewX(15deg);

skewY(angulo) - Inclina o elemento no eixo Y.

transform: skewY(25deg);

matrix(a, b, c, d, e, f) - Aplica uma transformação 2D combinada usando uma matriz. 

transform: matrix(1, 0, 0, 1, 0, 0);
